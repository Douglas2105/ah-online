# Приморско-Ахтарск: интерактивный гид у моря
Создать единый цифровой портал (веб-сервис) для жителей и гостей города Приморско-Ахтарска, который в реальном времени предоставляет всю необходимую информацию для комфортного отдыха и жизни у моря.  Главная цель: стать официальным или самым популярным онлайн-помощником для планирования досуга, повышения туристической привлекательности города.
